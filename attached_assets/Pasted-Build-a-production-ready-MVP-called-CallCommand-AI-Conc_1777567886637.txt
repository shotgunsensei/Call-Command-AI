Build a production-ready MVP called CallCommand AI.

Concept:
CallCommand AI is a mobile-first SaaS web app that turns phone call recordings into structured business intelligence: transcripts, summaries, action items, sentiment, priority, tags, follow-up tasks, and exportable records.

Brand direction:
Professional, dark-mode-first, cyber-technical, clean SaaS interface. Use the Shotgun Ninjas Productions style subtly: red accents, black/charcoal background, sharp card UI, modern command-center feel. Do not make it cartoonish.

Core stack:
- React + TypeScript frontend
- Node.js/Express backend OR FastAPI if easier
- PostgreSQL database
- Drizzle or Prisma ORM
- Stripe-ready subscription billing
- OpenAI-compatible AI integration
- File upload support for MP3, WAV, M4A, MP4
- Mobile-first responsive layout

Primary user flow:
1. User signs up / logs in.
2. User uploads a call recording.
3. App transcribes the audio.
4. App analyzes the transcript.
5. App returns a structured call intelligence report.
6. User can save, search, export, and send the report.
